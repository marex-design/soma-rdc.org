# SOMA – Site officiel

SOMA est un site web statique servant de page d’attente (Coming Soon).  
Il annonce le lancement prochain de la plateforme SOMA et assure une présence web simple, rapide et accessible.

---

## Accès au site

- URL : https://www.soma-rdc.org
- Hébergement : Vercel
- Certificat SSL : activé automatiquement

---

## Technologies utilisées

- HTML5
- CSS3 (intégré directement dans le fichier HTML)
- Hébergement statique via Vercel

Aucun framework ni dépendance externe n’est utilisé.
